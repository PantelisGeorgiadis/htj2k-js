# htj2k-js
Javascript HTJ2K Decoder for Node.js and Browser.

### Install
#### Node.js

	npm install htj2k-js

#### Browser

	<script type="text/javascript" src="https://unpkg.com/htj2k-js"></script>

### Build

	npm install
	npm run build
