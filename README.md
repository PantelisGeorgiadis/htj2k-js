# htj2k-js
High-Throughput JPEG 2000 (HTJ2K) decoder for Node.js and browser.

### Install
#### Node.js

	npm install htj2k-js

#### Browser

	<script type="text/javascript" src="https://unpkg.com/htj2k-js"></script>

### Build

	npm install
	npm run build

```js

```
